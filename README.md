# Explainable AI Based Predictive Maintenance and Anomaly Detection

## Overview

This project develops an Explainable AI based Predictive Maintenance Framework using Machine Learning and Deep Learning techniques on the AI4I 2020 Dataset.

## Objectives

* Predict machine failures before breakdown
* Detect abnormal machine behavior
* Explain model predictions using Explainable AI
* Evaluate robustness under noisy conditions
* Handle class imbalance using SMOTE

## Dataset

AI4I 2020 Predictive Maintenance Dataset

## Methodology

1. Data Preprocessing
2. One-Hot Encoding
3. Train-Test Split
4. Standard Scaling
5. Logistic Regression
6. Random Forest
7. Neural Network (MLP)
8. Gaussian Noise Injection
9. SMOTE Oversampling
10. SHAP Analysis
11. LIME Analysis
12. Autoencoder Anomaly Detection

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* TensorFlow / Keras
* SHAP
* LIME
* Matplotlib
* Seaborn

## Key Features

* Predictive Maintenance
* Explainable AI
* Anomaly Detection
* Noise Robustness Analysis
* Class Imbalance Handling

## Results

* Random Forest achieved the best overall performance.
* SHAP and LIME successfully explained machine failure predictions.
* Autoencoder effectively detected anomalies.
* The framework demonstrated robust performance under noisy industrial conditions.
